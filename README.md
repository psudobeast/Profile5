# Profile assignment 5

Overview
This project is a simple profile management application built with .NET MAUI. It allows users to input and save their profile information, including Name, Surname, Email Address, Bio, and a Profile Picture. The profile data is stored locally in JSON format within a text file, enabling persistence across app sessions.
Features

Profile Fields:

Name
Surname
Email Address
Bio


Profile Picture:

Add or change profile picture using device gallery
Profile picture is stored as Base64 within the profile JSON


Persistence:

Profile data is saved locally in JSON format
Data is loaded automatically when the app opens


UI Design:

Modern, clean interface with rounded borders and shadows
Easy to use with icons for each input field



Requirements

.NET MAUI SDK
Visual Studio 2022 or later
Platforms: Android, iOS, Windows, Mac
